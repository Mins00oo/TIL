# 1장 들어가기
### 자바가 지원하는 타입
- 인터페이스
- 클래스
- 배열
- 기본 타입

위 4개 중에서 인터페이스, 클래스, 배열은 참조 타입이라 한다. 즉, 객체인 반면, 기본 타입은 그렇지 않다. 

### 클래스의 멤버
- 필드
- 메서드
- 멤버 클래스
- 멤버 인터페이스
